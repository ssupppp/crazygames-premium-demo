# CrazyGames Premium Demo

Interactive website mockup demonstrating a Premium subscription model for CrazyGames platform.

## Project Overview
This is a Next.js demo built to showcase how a Premium subscription tier could work on CrazyGames, with a toggle-able interface showing Free vs Premium user experiences in real-time.

## Tech Stack
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Zustand** (state management)
- **Vercel** (deployment)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Project Structure
```
├── app/                 # Next.js pages (App Router)
├── components/          # React components
├── store/              # Zustand state management
├── data/               # Mock data files
├── types/              # TypeScript types
├── lib/                # Utility functions
├── docs/               # Project documentation
└── public/             # Static assets
```

- Toggle between Free and Premium user experiences
- 9 pages total (5 user-facing, 4 developer-facing)
- Conditional content rendering based on subscription status
- Mock game data and shop items
- Developer dashboard with revenue tracking
